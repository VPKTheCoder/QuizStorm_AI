## üöÄ Getting Started

### Using the Platform
Access QuizStorm AI at: **https://quizstorm-ai.netlify.app/**

### Creating Your First Quiz

1. **Enter a Topic**: Type any subject you want to learn about
   - Example: "History of Computers", "Python Programming", "Cell Biology"

2. **Choose Settings**:
   - **Number of Questions**: Minimum 3; Maximum 15
   - **Difficulty**: Easy, Medium, Hard, or Mixed
   - **Question Type**: Multiple Choice, True/False, or Mixed
   - **Timer**: Enable a timer if you want to time your answers

4. **Enter your AI API Key**: You need to enter an AI API key to generate the quiz. You can use any AI API key, but using the Google Gemini API key is recommended, as it provides users with the highest limits for free. 

5. **Generate**: Click "Generate Quiz" and watch it create

6. **Take the Quiz**: Answer questions at your own pace or according to the timer

7. **Review Results**: See your score and review explanations

## üí° Tips & Best Practices

### Creating Better Quizzes

**Topic Specificity**
- ‚úÖ Good: "The water cycle and precipitation"
- ‚ùå Too broad: "Science"
- ‚úÖ Good: "React hooks: useState and useEffect"
- ‚ùå Too vague: "Programming"

### Difficulty Level Guide

**Easy**
- Basic recall and recognition
- Straightforward questions
- Minimal distractors
- Perfect for beginners or quick reviews

**Medium**
- Application and understanding
- Some complexity
- Requires critical thinking
- Good for general assessment

**Hard**
- Analysis and evaluation
- Complex scenarios
- Subtle distractors
- Best for advanced learners

**Mixed**
- Varied difficulty
- Progressive challenge
- Realistic assessment
- Ideal for comprehensive testing

**Note**: The descriptions of these complexities are highly subjective and can change since the quiz questions are generated by AI.

### Quiz Management Tips

1. **Name Your Quizzes**: Use descriptive topics for easy retrieval
2. **Review Wrong Answers**: Learn from explanations to improve
3. **Retake Quizzes**: Saved quizzes can be retaken anytime
4. **Export for Sharing**: Download quizzes to share with study groups
5. **Track Progress**: Use quiz history to see improvement over time
